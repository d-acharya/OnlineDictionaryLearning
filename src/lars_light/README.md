# lars_light
a header-only implementation of Lars(least angle regression)

[Efron, Bradley, et al. "Least angle regression." The Annals of statistics 32.2 (2004): 407-499.](http://statweb.stanford.edu/~tibs/ftp/lars.pdf)

Usage
------------
```bash
mkdir build
cd build
cmake ..
make
./test_lars
```
